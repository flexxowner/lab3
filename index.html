Добро пожаловать на мою первую HTML страницу!

Это небольшой текст, который я написал с помощью HTML. HTML (HyperText Markup Language) является стандартным языком разметки для создания веб-страниц. Он используется для структурирования содержимого веб-страниц и определения их внешнего вида.

На этой странице мы использовали заголовок первого уровня (h1) для основного заголовка и параграфы (p) для текста. Это лишь малая часть того, что можно сделать с помощью HTML. С его помощью можно также добавлять изображения, ссылки, таблицы и многое другое.

HTML - это основа веб-разработки, и его изучение может открыть для вас много новых возможностей!
